GIT VÀ GITHUB

GiT là gì??

Git là một hệ thống quản lý source code giúp theo dõi sự thai đổi của mã nguồn trong quá trình phát triển phần mềm .Git cho phép nhiều người cùng làm việc với nhau trên một dự án mà không ảnh hưởng đến công việc của nhau.

Tải Git tại đây

Các tính năng chính của GIT

  1. Lưu trữ lịch sử commit
  2. Revert về vesion củ
  3. làm việc theo nhóm hiệu quả
  4. Hổ trợ tạo nhánh(Branch)

Github là gì?

Github là một nền tảng lưu trử mã nguồn trực tuyến , hoạt động dựa trên Git , nó cho phép lập trinhf viên tải mã nguồn lên ,cộng tác với nguời khác,theo dỗi lỗi , quản lý dự án một cách dễ dàng

Những điểm giống và khác nhau giữa Git và Github

Tiêu chí Git Github
Cách hoạt động Chạy trên máy tính, không cần internet. Hoạt động trên web, cần internet.
Chức năng chính Quản lý phiên bản mã nguồn. Lưu trữ mã nguồn, hỗ trợ làm việc nhóm.
Lưu trữ dữ liệu Trên máy tính cá nhân. Trên máy chủ GitHub
Cách sử dụng Dùng dòng lệnh hoặc phần mềm hỗ trợ. Giao diện web thân thiện.
Bảo mật Phụ thuộc vào thiết lập của người dùng. Hỗ trợ phân quyền, bảo vệ dữ liệu tốt hơn.
Giá cả Miễn phí, mã nguồn mở. Có gói miễn phí và trả phí với nhiều tính năng.

Cac lenh co ban trong git

  1. git init
  2. git clone
  3. git pull
  4. git push
  5. git fetch
  6. git commit -m ""
