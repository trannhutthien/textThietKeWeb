Tiêu chí Git Github
Cách hoạt động Chạy trên máy tính, không cần internet. Hoạt động trên web, cần internet.
Chức năng chính Quản lý phiên bản mã nguồn. Lưu trữ mã nguồn, hỗ trợ làm việc nhóm.
Lưu trữ dữ liệu Trên máy tính cá nhân. Trên máy chủ GitHub
Cách sử dụng Dùng dòng lệnh hoặc phần mềm hỗ trợ. Giao diện web thân thiện.
Bảo mật Phụ thuộc vào thiết lập của người dùng. Hỗ trợ phân quyền, bảo vệ dữ liệu tốt hơn.
Giá cả Miễn phí, mã nguồn mở. Có gói miễn phí và trả phí với nhiều tính năng.

Cac lenh co ban trong git

  1. git init
  2. git clone
  3. git pull
  4. git push
  5. git fetch
  6. git commit -m ""
